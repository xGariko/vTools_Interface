# vContacts
vContacts is a simple tool to convert excel data sheet into VCF card in order to save lot of contacts rapidly.

```shell
poetry install
```

To install all the libraries used

```shell 
poetry add package-name
```

---

```shell
poetry add --dev package-name 
```

To add package only in dev environment
